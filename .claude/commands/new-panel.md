Create a new panel for $ARGUMENTS:
1. create `components/panels/$ARGUMENTS-panel.tsx`
2. create `types/$ARGUMENTS.ts`
3. create `hooks/use-$ARGUMENTS.ts`
4. create route handlers if needed
5. create tests
6. register the panel
7. run verification
